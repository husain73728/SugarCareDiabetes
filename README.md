# 🩺 SugarCareDiabetes

An AI-powered healthcare platform for diabetes prediction and management.

---

## 🚀 Features

- 🔍 Diabetes risk prediction using ML model  
- 🥗 Personalized diet recommendations  
- 🏃 Exercise and fitness plans  
- 👨‍⚕️ Doctor consultation interface  
- 💊 Pharmaceutical support  
- 🤖 AI chatbot assistance  
- 🌐 Multilingual support  

---

## 🧠 Tech Stack

- HTML, CSS, JavaScript  
- Machine Learning (Diabetes Prediction Model)  
- JSON-based model integration
